## HTML

### Tags (Sintase)

Estrutura:

```text
openning              closing
  tag      content      tag
  ┌┴┐┌────────────────┐┌─┴─┐
  <p>Lorem ipsum dolor.</p>
  └────────────────────────┘
          Element
```

Exemplos:

- Cabeçalho: `<title>`, `<link>`, `<style>`, `<meta>`
- Texto: `<p>`, `<b>`, `<i>`, `<h1-6>`
- Lista: `<ul>`, `<ol>`, `<li>`, `<dl>`, `<dt>`, `<dd>`
- Hiperlink: `<a>`
- Tabela: `<table>`, `<tr>`, `<th>`, `<td>`
- Imagem: `<img>`
- Mais exemplos: [HTML Reference - MDN](https://developer.mozilla.org/en-US/docs/Web/HTML/Element)

### Atributo de tag (Sintase)

Estrutura:

```text
     attribute
   ┌─────┴────┐
<p lang="pt-BR">Lorem ipsum dolor.</p>
```

Exemplos:

- Global: `lang`, `id`, `class`
- Específico:
  - `<img>`: `src`, `alt`
  - `<ol>`: `start`, `type`
  - `<table>`: `border`
  - `<meta>`: `name`, `content`

Código:

```html
<meta name="description" content="A description of the page" />
```

### Hyperlink

Estrutura:

```text
hyperlink = content + url

content = text, image...
```

```html
<a href="url">content</a>
```

Exemplo:

```html
<a href="http://www.ifpb.edu.br/">ifpb</a>
```

Resultado:

<HyperText>
  <a href="http://www.ifpb.edu.br/">ifpb</a>
</HyperText>

### Estrutura básica

```html
<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8" />
    <title>Bem-vindos</title>
  </head>
  <body>
    <h1>Olá, turma!</h1>
    <p>Meu primeiro parágrafo</p>
  </body>
</html>
```

Output:

<HyperText>
  <h1 style={{color: "#606c71"}}>Olá, turma!</h1>
  <p>Meu primeiro parágrafo</p>
</HyperText>

### Imagem

Estrutura:

```html
<img src="url" alt="text alternative" />
```

Exemplo:

**index.html => image.png**

```text
/
└── var
    └── www
        └── site
            └── pages
                ├── index.html
                └── img
                    └── image.png
```

```html
<img src="img/image.png" alt="Image Description" />
<img src="/var/www/site/pages/img/image.png" alt="Image Description" />
```

Resultado:

<HyperText>
  <img src="/imgs/hypertext/ifpb-logo.png" alt="Image Description" />
</HyperText>
